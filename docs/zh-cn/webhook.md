# 使用Notr测试webhook

Notr的另外个主要应用领域是webhook，我们在实际开发当中需要用到第三方服务，第三方服务为了给予开发者更多的控制，于是就有了webhook，开发者只需要在第三方平台填写api，指定时间发生时，第三方平台会调用该api。第三方服务毫无疑问是部署在云上的，所以需要开发者每次都将应用发布到公网，然后再进行测试。

使用Notr之后，开发者不需要通过将应用发布到云服务器，在本地即可调试。
